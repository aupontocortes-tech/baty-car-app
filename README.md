# OpenALPR Car Plate App

Aplicativo React + Node.js para leitura de placas usando OpenALPR.

## Instalação

```
npm install
```

## Desenvolvimento

```
npm run dev
```

Servidor: `http://localhost:5000`  
Cliente: `http://localhost:3000`

Se a porta 3000 estiver ocupada, finalize o processo atual ou altere a porta do cliente.

## Build

```
npm run build
```
